# jet_print_playground

The playground app for the `jet_print` library. It consumes the library through
its public entry point only, showing the interactive report designer with worked
samples (invoice, labels, barcodes, menu, nested lists, packing slip, payroll)
inside a shadcn-themed shell with a light/dark toggle.

Runs on macOS, Windows, Linux, web, and iOS/Android — CI builds every target on
each push. See the [workspace README](../../README.md) for install, run, and
test instructions, and the library's
[Platform support](../../packages/jet_print/README.md#platform-support) table for
per-platform CI coverage.
